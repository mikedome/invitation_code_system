# 邀请码生成系统开发计划

## 系统架构
- **前端**：Vue3 + Vite + Element Plus
- **后端**：Node.js + Express
- **数据库**：MySQL

## 实现步骤

### 1. 项目初始化
- 创建前端Vue3项目（使用Vite）
- 创建后端Node.js项目
- 配置项目依赖

### 2. 数据库设计
- 设计`invitation_codes`表，包含字段：
  - `id`：主键，自增
  - `code`：邀请码（唯一）
  - `employee_id`：员工编号
  - `generated_at`：生成时间
  - `status`：状态（已使用/未使用）

### 3. 后端实现
- 配置数据库连接
- 实现邀请码生成逻辑：
  - 员工编号（4位，如0001）+ 随机字符串（12位），总长度16位
  - 确保邀请码唯一
- 实现API接口：
  - `/api/generate`：生成邀请码
  - `/api/history`：查询历史记录
  - `/api/validate`：验证邀请码

### 4. 前端实现
- 创建邀请码生成页面：
  - 员工编号输入框
  - 生成按钮
  - 邀请码显示区域
- 创建历史记录页面：
  - 表格展示所有生成的邀请码
  - 筛选和分页功能
- 配置Axios请求

### 5. 部署配置
- 编写部署脚本
- 配置环境变量
- 测试系统功能

## 技术要点
- 邀请码生成算法确保唯一性
- 数据库索引优化查询性能
- 前端响应式设计适配不同设备
- 错误处理和日志记录
